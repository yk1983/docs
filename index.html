커피와 개발자

웹 프로젝트를 하기위해 가장 쉽게 따라하면서 배우는 웹 스터디 !!
HTML5/CSS3
HTML5는 HTML의 완전한 5번째 버전으로 월드 와이드 웹 (World Wide Web)의 핵심 마크업 언어이다.
JavaScript
자바스크립트(영어: JavaScript)는 객체 기반의 스크립트 프로그래밍 언어이다. 이 언어는 웹 브라우저 내에서 주로 사용하며, 다른 응용 프로그램의 내장 객체에도 접근할 수 있는 기능을 가지고 있다. 또한 런타임 환경과 같이 서버 프로그래밍에도 사용되고 있다.
jQuery
jQuery는 MIT 라이선스를 가진 자유 오픈 소프트웨어이다. jQuery의 문법은 코드 보기, 문서 객체 모델 찾기, 애니메이션 만들기, 이벤트 제어, Ajax 개발을 쉽게 할 수 있도록 디자인되었다. 또한, jQuery는 개발자가 플러그인을 개발할 수 있는 기능을 제공한다.
Java
자바(Java)는 객체 지향적 프로그래밍 언어이다. 현재 웹 애플리케이션 개발에 가장 많이 사용하는 언어 가운데 하나이고, 모바일 기기용 소프트웨어 개발에도 널리 사용하고 있다.
SQL
SQL(Structured Query Language)은 관계형 데이터베이스(RDBMS) 관리 시스템에서 자료의 검색과 관리, 데이터베이스 스키마 생성과 수정, 데이터베이스 객체 접근 조정 관리를 위한 언어입니다.
SpringBoot
스프링은 자바 기반의 웹 어플리케이션을 만들 수 있는 프레임워크이며, 스프링 부트(Spring Boot)는 스프링(Spring)을 더 쉽게 이용하기 위한 도구라고 볼 수 있습니다.
thymeleaf
타임리프는 흔히 View Template(뷰 템플릿)이라고 부르며, HTML태그를 기반으로하여 th:속성을 이용하여 동적인 View를 제공한다. 웹에서 가장 기본이되는 HTML로 진입장벽이 낮고 쉽게 배울 수 있다는 장점이 있다.
TOAST UI
TOAST UI는 NHN에서 개발한 오픈소스로, Grid, editor, chart, calendar 등을 이용할 수 있다.
ETC
프로그래밍 언어의 기본기 및 개발 스킬 이외의 자기개발을 위한 학습을 할수 있는 내용입니다.
